# Linux & Cloud Administration Lab

A hands-on learning repository documenting Linux system administration,
server hardening, cloud infrastructure, automation, and networking.

## Objectives

- Learn Linux administration through practical labs
- Understand Linux security fundamentals
- Deploy and manage cloud infrastructure
- Build production-like server configurations
- Learn automation using Bash
- Learn system services using systemd
