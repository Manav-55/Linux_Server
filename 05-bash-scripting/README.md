# Bash Scripting

## Objective

Learn the fundamentals of Bash scripting by automating common Linux administration tasks. This section focuses on writing reusable, maintainable, and practical shell scripts rather than executing commands manually.

## Concepts Covered

* Shebang and script execution
* Variables and command substitution
* User input
* Conditional statements
* Loops
* Arrays
* Functions
* File operations
* Logging
* Error handling
* File compression
* Automatic cleanup

* Lab 12 – Backup Automation Project

## Outcome

By the end of this module, a complete backup automation script is developed that:

* Creates timestamped backups
* Compresses directories
* Logs script execution
* Handles common errors
* Removes outdated backups automatically

This module establishes the scripting foundation required for Linux administration, automation, DevOps, and cloud engineering.
